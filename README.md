# Retirement Planning Simulator

## Overview
A Python-based financial modeling tool that helps users plan for retirement by simulating different investment and withdrawal scenarios. The program provides four key financial calculations related to retirement planning.

## Features

### 1. **Safe Withdrawal Amount Calculator**
Calculates the maximum annual withdrawal amount that can be sustained over a fixed retirement period without depleting the portfolio, considering compound growth.

### 2. **Fixed Investment Growth Calculator**
Projects the future value of an investment with a fixed annual rate of return using compound interest formulas.

### 3. **Variable Investment Growth Calculator**
Simulates investment growth with changing annual rates of return, providing year-by-year balance tracking.

### 4. **Retirement Fund Duration Simulator**
Determines how long a retirement fund will last given a fixed annual withdrawal amount and investment growth rate.

### Requirements
- Python 3.8 or higher
- No external libraries required

   - `RetirementSimulator.py`
   - `MaxWithdrawal.py`
   - `FixedInvestor.py`
   - `VariableInvestor.py`

2. Run the program:
